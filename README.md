Films - данный сервис предоставляет пользователю краткую информацию о фильме на запрашиваемую дату.

EmailSenderAPI-main - обеспечивает отправку электронных писем с HTML-контекстом через SMTP-сервер.

Film-Email - веб-платформа, которая объединяет собственное API и API одногруппника.

Логика взаимодействия:

-пользователь вводит дату и email;

-система отправляет запрос к API 1 (собственный API);

-формирование ответа с данными;

-отправка на почту через API 2 (API одногруппника);

-пользователь получает письмо на почту;
